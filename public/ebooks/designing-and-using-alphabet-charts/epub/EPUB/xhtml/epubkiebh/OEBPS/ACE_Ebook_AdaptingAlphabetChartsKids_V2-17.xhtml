Alternative Alphabet Arrangements

Encoding is a way of helping someone to point to letters who cannot manage selecting between 26 different letters on an alphabet chart. Encoding groups letters together, reducing the number of target areas on a page.

Coding is another option that can be considered. Here the individual uses a smaller chart of codes to communicate what is effectively the grid reference of the letter.

You can read much more about both of these options in Chapter 8 Arranging Letters on an Aphabet Chart.
