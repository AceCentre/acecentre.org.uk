While an E-tran frame (or similar) is most commonly used for eye pointing, there are alternatives. Someone could eye point to a chart presented on a recipe book / document stand or easel file or make use of a whiteboard (see Beatrice’s communication system for an example of this). Such tools need to be positioned with care to ensure the communication partner can still easily follow the eye movements.

An alternative way of displaying an alphabet chart for eye pointing – on an easel file

If someone is very familiar with using an eye pointing alphabet chart, it may be that they can communicate with eye movement without actually using the chart at all. This only works with a familiar and experienced communication partner as both will need to know the chart well enough to visualise it without actually seeing it, so it would be important to have a copy of the chart available when communicating with others.
