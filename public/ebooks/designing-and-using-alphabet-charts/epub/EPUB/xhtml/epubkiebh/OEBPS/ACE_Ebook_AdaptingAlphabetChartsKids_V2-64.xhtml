Communication Books Accessed by Eye PointingCommunication books can also be developed for use with eye pointing. One such resource is Speakbook.

Speakbook was created by Patrick Joyce who had Motor Neuron Disease. It is a fully customisable communication book template that is designed to be used with eye pointing. It can be downloaded from Ace Centre’s website here. As well as the version illustrated below, there is a black and white option that may be more suitable for someone who is colour blind. A growing number of languages are supported.

 

The Speakbook is held by the communication partner. There is a central window cut into all pages through which eye contact is made, enabling the communication partner to more easily follow somenoe’s eye movements. It uses a style of encoding to give access to up to sixteen messages on each page. Messages are handwritten using a dry erase pen onto the laminated sheets, allowing it to be easily updated. The messages are duplicated on the front and back to help the communication partner follow what is being looked at.

The first page of the book is a Home Page, which is effectively an index page giving access to topic pages of an individual’s choosing. The book also contains a spelling page that can be accessed from the Home Page and every topic page.

This is a video of Patrick, who created the Speakbook, with David, explaining how the Speakbook works:

Angie demonstrates using the Speakbook. She is eyepointing to the block containing the messages ‘Go to bed’ and ‘Get up’. To clarify which option she is communicating, she will then eyepoint to the coloured dot that matches the colour of the stripe her target message is written on.

The Speakbook contains an alphabet page. To select a letter or number, you would first look at the block that contains your target letter or number. You would then clarify which letter or number you are selecting by looking at the block whose colour matches that of your target letter or number. It is very similar to the encoded eye pointing charts discussed above.
