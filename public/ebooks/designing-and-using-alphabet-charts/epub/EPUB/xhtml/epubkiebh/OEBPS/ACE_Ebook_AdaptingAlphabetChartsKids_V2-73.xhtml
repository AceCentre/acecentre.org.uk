Key Communication Partner Skills – a Summary

• Understand how someone communicates ‘yes’ and ‘no’

• Allow enough time

• Learn how the communication system works

• Where appropriate, write down each letter selected to keep track of the conversation

• Establish whether it is ok to attempt to predict what is being communicated, and if so, always check the guess is correct

• Stay calm if something goes wrong

• Avoid the trap of double checking that each letter or word is correct before moving on to the next letter, but take action if they have gone wrong

• Where relevant, check whether someone has more to say before putting down the chart

• Only speaking for someone if asked

﻿Module four of the freely available online learning resources at aacscotland.org.uk provides more information about being a good communication partner and offers some excellent downloadable resources.

Other Considerations for Successful Communication

It is worth thinking about the environment in which the communication is taking place. A quiet place with fewer background distractions is often a better setting for a conversation.

It can help if the communication partner faces the person they are talking to, as this will help them to read their body language and facial expressions. Having said that, it is sometimes the case that someone with a visual difficulty will find it easier if the communication partner stands on one side or another.

Communicating with an alphabet chart can sometimes be easier one-on-one, or in smaller rather than larger groups of people.

Some choose to use key words or short sentences to improve the efficiency of their communication. If comfortable using them, simplified spellings can also speed things along e.g. ‘u’ for ‘you’ or ‘wen’ for ‘when’. However, this is very much down to personal preference.
