﻿Coded Layouts

Coded layouts make use of a grid reference system. Letters (and numbers / words / phrases) are each given a grid reference. It is this reference rather than the letter itself that is communicated by the individual. It’s a bit like using a road atlas - the location of a place is identified by its grid reference.

Instead of using a chart of twenty-six individual letters, with a coded layout, the individual only has to select from a limited number of options to deliver the grid reference. Coded layouts therefore involve two charts - a main chart displaying the alphabet and a separate chart displaying the information needed to communicate the location of the letter (and number / word / phrase).

The main chart needs to be displayed so that both can see it. The separate chart with the location information needs to be easily accessed by the individual.

In this example, the first chart shows the letters on offer. The second smaller chart is the one that the individual points to. A similar chart is available to download from acecentre.org.uk.

To communicate the letter ‘P’ on the main chart, the individual would first select the number ‘3’ on their chart of codes, as ‘P’ is in the third row. They would then select the colour ‘blue’ as ‘P’ is in the blue column.

This chart shows the letters and messages on offer.

The smaller chart is used to indicate the code that corresponds to the target letter / message on the first chart.
