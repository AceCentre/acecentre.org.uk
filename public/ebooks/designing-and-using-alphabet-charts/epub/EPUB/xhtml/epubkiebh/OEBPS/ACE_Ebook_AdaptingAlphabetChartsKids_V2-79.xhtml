Communication Book Templates - Freely Available

You can download templates for a number of different communication books from Ace Centre https://acecentre.org.uk/product-category/alphabet-books/

You can download a template for making a communication tabbed flip book from the ALS Augmentative Communication Program area of the Boston Children’s Hospital website https://www.childrenshospital.org/centers-and-services/programs/a-_-e/als-augmentative-communication-program/protocol-of-assessment-considerations/quick-access-and-low-tech-tools

Speakbook is a freely available communication book template that is designed to be used with eye pointing. It uses an encoded keyboard layout and provides access to useful words and phrases. Developed by Patrick Joyce, this can now be downloaded from Ace Centre acecentre.org.uk/project/speakbook/

Communication Books - Resources available for purchase

Ability World    www.ability-world.com sell a range of folders which can be used with a communication book

﻿Making Your OwnWhile there are a lot of resources available to download or purchase, you may well find you want to make your own.

If you are creating resources for someone else in a language you are unfamiliar with, it is worth working with a native speaker of that language. As discussed in Frequency Layouts, do bear in mind that if you are creating a frequency layout for another language, the frequency of use of letters for that language will differ to their frequency of use in English.

The simplest way of creating your own alphabet chart is to make a table

within word processing or spreadsheet software. Similarly, when creating a communication book, you could either write messages in a notebook or use word processing software.

If you have access to specially designed﻿ resource making software (most likely this will be through a Speech and Language Therapist), you may find this offers a bit more flexibility when it comes to developing an alphabet chart. Some software also includes templates of alphabet charts. Examples of such software in the UK includes:

• The Boardmaker range of software e.g. available from tobiidynavox.com

• Communicate in Print 3 e.g. available from widgit.com

• Matrix Maker Plus e.g. available from inclusive.co.uk

Such software is particularly valuable if you need to create a communication book or chart with symbol support.
