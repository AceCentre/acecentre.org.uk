Thinking about PositioningWhile not a design issue, the positioning of a chart may also help visual access. Placing the chart on a wedge or document stand, for example, might help someone to see it better. If someone is using a wheelchair (or other) mounting system, it might also be possible to attach the alphabet chart to this. This should be discussed with the suppliers of the mounting system.

A visual field issue might also mean that the positioning of the alphabet chart affects how easy it is to see it.

Tactile AlphabetIt is possible to produce an alphabet chart in braille if required, and an example of such a chart is shown below.

Braille keyboard stickers are available from the RNIB shop. Alternatively, you could produce braille letters using the 6dot™ Braille Label Maker™ which is available from Logan Technologies. When constructing an alphabet chart in braille, make sure the printed letter is included alongside the braille letter so that communication partners can understand what is being pointed to.
