Unlike with ABC and QWERTY layouts, there is no single standard frequency layout. The letter arrangement chosen will vary according to the access method used. Beatrice for example, makes use of a customised frequency layout within each block of her whiteboard system.

You will also different versions of frequency layouts even within access methods. If someone is using a frequency layout within voice output and / or recording software, you may want to check that the letter layout is consistent with their paper-based chart to reduce confusion.

Note that the frequency ratings for each letter will vary according to the language spoken. The letter ‘Z’ is used much more frequently in Polish than it is in English, for example. If you find yourself having to create a frequency layout for another language, you will need information about the frequency of use of letters for that language. There is some information about this here on Wikipedia that may be a useful starting point, although it is not comprehensive.

If a frequency approach is used, it may prove more efficient to have separate alphabet charts for each language.

In Summary - Frequency Layout

Why consider a frequency layout?

A frequency layout has been designed to make selecting letters more efficient which should speed up communication.

What access methods does it suit?

A frequency layout can work well with pointing and listener mediated scanning, so long as the chart has been designed with the particular access method in mind.

What are the disadvantages?

A frequency layout has to be learned by an individual and / or their communication partners as it is unlikely that all will have encountered one previously.

While a frequency layout chart for listener mediated scanning can be downloaded from acecentre.org.uk, there isn’t a standard layout (as there is with QWERTY for example) so you will need to check the paper-based chart is consistent with any onscreen frequency keyboards used elsewhere.

The frequency of each character is language dependent. Therefore a separate chart will probably be required if the person needs to communicate in another language.
