Standard alphabet charts tend to be designed to be used with an index finger, although any part of the hand or indeed body can be used. They can be a good option to consider if writing is difficult.

It may be worth keeping copies of alphabet charts in different locations so that one is always to hand when needed.

Try and keep a note of frequently used messages or phrases. Rather than spelling the same thing repeatedly, such messages can be delivered more efficiently. See Chapter 10 for more information.
