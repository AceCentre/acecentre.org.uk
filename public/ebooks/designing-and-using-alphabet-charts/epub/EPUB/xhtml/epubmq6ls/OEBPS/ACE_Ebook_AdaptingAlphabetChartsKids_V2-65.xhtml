﻿Communication Books Accessed by Listener Mediated ScanningJust as alphabet charts can be accessed by listener mediated scanning, so too can communication books. The communication partner offers each message in turn by pointing to the message and / or speaking it aloud. The person indicates “yes” when they point to and / or say their target message.

You could also create your own eye pointing communication book and display it in an easel file / presentation portfolio like in this example. Easel files tend to be available from office stationery retailers. Alternatively, a customised eye pointing book could be placed on a recipe book or tablet stand. Like with an easel file, these angle the pages, making it easier for the communication partner to follow the eye pointing.
