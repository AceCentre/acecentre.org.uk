﻿Alphabetical Layout - ABC

When letters are placed in alphabetical order, it should feel very familiar to the individual. An ABC layout can also help communication partners work out what letter is being pointed to - if someone’s finger / other body part is obscuring the letter itself, the communication partner can easily work out what is being pointed to by looking at the adjacent letters.

This is an example of an ABC layout alphabet chart that is designed to be accessed by pointing. It is similar to the one used by Anjali.

Arranging letters in alphabetical order has the advantage of minimising the time needed to learn where the letters are on the chart. For someone using eye pointing, an ABC layout can keep things simpler for both the person using the chart and the communication partner. Similarly, for someone using listener mediated scanning, an ABC layout can mean they don’t have to watch and / or listen quite so intently, particularly when they are learning to use the system.

However, as with a QWERTY layout, a potential disadvantage is that the ABC layout was not designed to be an efficient way of selecting letters in the way that frequency layouts have been.

In Summary - ABC

Why consider ABC?

An ABC layout should feel familiar to everyone as it taps into our knowledge of the alphabet. This can help both the person using the chart and their communication partner.

What access methods does it suit?

ABC can suit all access methods.

What are the disadvantages?

Using an ABC layout can feel a bit strange initially for someone who is very familiar with a QWERTY layout. An ABC layout is not optimised for speed and efficiency.
