Encoded charts can be set up in different ways. Some people prefer to use thick coloured borders round the groups of letters rather than dots, others identify the groups using shapes or numbers. But the principle remains the same.

This encoded alphabet chart (available to download from acecentre.org.uk)uses thick coloured borders around the groups of letters as well as coloured dots in the centre of each group. Note that this example also makes use of upper rather than lower case letters which is a matter of personal preference. See Chapter 9 for more discussion around such design issues.
