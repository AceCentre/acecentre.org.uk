If using hands is difficult, people occasionally choose to access an alphabet chart using head movement. This can either be done indirectly using an LED light pointer or directly using a headpointer. These tools can be used with any type of alphabet chart, including encoded or coded charts.

A fairly discreet option is to attach ﻿an LED light pointer to a pair of glasses, a head band or a hat. The person then gently moves their head to direct the light beam to highlight letters or words on a chart. As discussed above, LED pointers are designed to help presenters to highlight key points when delivering presentations and are therefore easy to source, although the head or glasses mounting is less standard. As far as we know, a pointer with a purpose built clip for use with glasses or a cap can only be purchased from a company in America store.lowtechsolutions.org
