﻿Frequency Layouts

A frequency layout is an alternative to QWERTY and ABC, but is not universally standard in the same way. With a frequency layout, letters that are used frequently in English such as ‘E’ or ‘T’, are placed where they are quicker / easier for you to select, and letters that are used infrequently, such as ‘Q’ or ‘Z’, are placed further away.

This approach tends to be used with listener mediated scanning, as in the chart below. For more information, and to see a video of this chart in use, see the section on Using a Frequency Layout with Listener Mediated Scanning in Chapter 5.

The patented FITALY keyboard layout

The FITALY layout has been designed to minimise the amount of movement required when spelling words using a single finger. The most frequently used letters are towards the centre of the chart, whilst less frequently used letters are around the edges. Although invented and patented by Jean Ichbiah for use with a computer or handheld device, the keyboard layout could also work well on a paper-based chart.

While a frequency layout tends to be used with listener mediated scanning, it can be used with other access methods. The FITALY keyboard is a rare example of a frequency layout designed for people who point using a finger. See fitaly.com for more information.

Keyboard layouts designed for joystick use may also be of interest as, like the FITALY layout, they are designed to be accessed by a single pointer rather than ten fingers. A keyboard layout optimised for a joystick will aim to minimise amount of movement the pointer must make which can result in a more efficient selection of letters.
