Examples of PODD Books

Score is a paper and power-based vocabulary that is available from jabbla.co.uk. There are communication charts and a binder which complement the Score vocabulary package in MindExpress 5 communication software. It is designed to be accessed by pointing.

The Super Core communication book is based around the Super Core vocabulary package found within Grid 3 communication software. It is a symbol-based communication book and is designed to be accessed by pointing or listener mediated scanning. It is available from thinksmartbox.com.

Other Resources for Making a Communication Book

There are a number of freely downloadable communication book templates available on the Ace Centre website – see acecentre.org.uk. While these are largely text based, they do include templates for creating communication books based around auditory scanning, including the pragmatic-based auditory scan book discussed in the section above.

For those using the Proloquo2Go app by AssistiveWare, there are instructions on how to create a complementary paper-based communication book on their website – see assistiveware.com/blog/making-a-paper-based-aac-book. These could also provide inspiration for adapting other vocabulary packages within communication software.

Ability World sells lots of A5, A4 and A3 folders that can be used to make communication books. They have practical features like handles and velcro fastenings. They also have creative solutions like small books that can be worn on a belt or lanyard, waist-mounted books and totes. See ability-world.com.

Courses on PODD are available in the UK, and are a great way of learning more about this way of communicating. You can find a list of UK-based trainers and information about forthcoming courses at podd.dk (click on the Union Jack flag). The templates themselves can be purchased from inclusive.co.uk.
