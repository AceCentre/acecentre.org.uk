Consider the role of an alphabet chart

Getting started with symbol-based AAC can feel daunting! This section gives you some ideas about how you might begin. It includes examples of communication charts that you can download from ACE Centre’s website (acecentre.org.uk) and use to get started. However, you will soon want to adapt these and make your own.

The key to getting started is to find a way of bringing AAC into daily activities that are relevant to the individual and their environment, but more importantly, that are motivating and fun.

Getting started with symbol-based AAC is also about using the chart yourself and being a good model of how symbol-based communication works! You can read much more about this in the next chapter.

Every Journey Begins with One Step…

If you start by trying to use AAC all the time, it is likely to feel overwhelming and something that is better left until there is more time available. However, as the inevitable issues and challenges arise, the moment can never seem right for such a seemingly big undertaking.

Instead, start today, but start small. If you start small, you can build on success over time. While the ultimate goal is for AAC to be integrated into daily life, this will take time and we all need to be realistic about this. A little AAC is a lot better than no AAC!

One way that might help you get started is to set yourself a five minute challenge. Is there a five minute window in your day where you might be able to introduce AAC alongside a fun activity?

Choose one activity and get started.

Find an activity that feels right for the person who will be using the chart, yourself and home / the environment you are in.

Make sure the activity is something the person is interested in and enjoys.
