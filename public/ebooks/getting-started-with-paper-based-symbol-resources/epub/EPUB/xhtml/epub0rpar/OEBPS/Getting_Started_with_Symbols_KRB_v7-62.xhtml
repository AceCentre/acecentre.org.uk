Use of Colour

Colour can be used in lots of different ways in paper-based resources. It can be used on the page background, the symbol background and / or on the border of the cell surrounding the symbol. Colour can be used to help people find information, to help people see the symbols better and / or to give information about the type of symbol it is (noun, verb, adjective, etc). Sometimes, like when using a personalised background image, it is just used because someone enjoys the colour.

In this example, a busy set of symbols is separated out using blocks of background colour. As someone flicks through their communication book, they are also helped in finding the ‘Places I go’ page as it is coloured purple. Other topic pages have different background colours.
