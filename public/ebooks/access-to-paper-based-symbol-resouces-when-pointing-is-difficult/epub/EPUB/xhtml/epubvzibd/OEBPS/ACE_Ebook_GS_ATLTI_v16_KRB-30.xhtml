﻿Presenting Symbols when Eye Pointing

Number of Items

One of the first considerations when setting up an eye pointing system is how many symbols to present at any one time on the E-tran frame or chart. This is a real balancing act between language and communication needs, and eye pointing ability. If there are too many options, it can be difficult for the communication partner to interpret the direction of gaze.

When symbols are presented on an E-tran frame or similar, people tend to limit the number of symbols (or groups of symbols) presented in order to limit the number of different eye pointing movements that will be required to make selections. The number presented will vary, depending upon the individual’s eye pointing skills, visual ability and their language needs, but people tend to present between two and eight options at any one time.

If many more symbols than this are required, this can be achieved using encoding or an EyeLink style chart. Both of these options are explored below.
