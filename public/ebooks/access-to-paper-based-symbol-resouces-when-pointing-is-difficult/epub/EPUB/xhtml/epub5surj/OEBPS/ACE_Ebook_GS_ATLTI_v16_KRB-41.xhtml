Encoding can initially feel a bit complicated, both for the communicator and for the communication partner.

It is often best to start small, perhaps using just two colours or numbers, within a fun and motivating game or activity. Modelling how to use the encoding by using it yourself is always a great start. Once someone is confidently communicating using two colours, you can then increase the number of symbols that are used in each block.
