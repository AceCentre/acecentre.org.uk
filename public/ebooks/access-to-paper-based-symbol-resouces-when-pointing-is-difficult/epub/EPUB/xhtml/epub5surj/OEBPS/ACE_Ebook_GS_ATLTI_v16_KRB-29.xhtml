Easel File

As well as acting as storage for eye pointing charts, an easel file / binder is another way of presenting material for eye pointing communicators. The nature of the file ensures that the charts are presented at a suitable angle, although positioning is a little trickier than with an E-tran frame. But their main advantage is that the charts can be presented handsfree. As with the laminated eye pointing charts, using an Easel file can also speed communication as there is no need to attach or re-attach symbols to a frame whenever there is a new topic of conversation or activity. Easel files are available from some of the large office stationery suppliers

Alternatively, alongside the Look2Talk guide, Ace Centre produces a more robust easel file that can be purchased as a standalone item – see acecentre.org.uk/resources/look2talk.

A recipe book or document stand could also be used.

Section 1 - E-tran Frames and Alternatives

Another way of making an eye pointing chart is to laminate a whole piece of paper with only a central window removed. In this case, matt laminate is preferable. A little less cutting is involved but you can only see the communicator’s eyes through the central window.

A collection of either type of laminated eye pointing charts can kept together to form the basis of a communication book. You might bind them together using loose rings or treasury tags, or store them in a file. The advantage of this system is that the communication partner does not have to add and remove symbols from a Perspex® E-tran frame each time there is a change of activity or topic of conversation. In addition, the symbols stay in the same place each time they are presented which can help with learning. See Chapter 11 Next Steps - developing a Communication Book for more information.
