Chapter 6

﻿﻿Coded Access

In this chapter:

• Be introduced to coded access

• Consider different ways of coding symbols

Coded access is another way of increasing the number of symbols available on a page beyond the number that someone is able to point to directly.

Coded access describes an access method where symbols are effectively given a grid reference that the individual then communicates.

Coded access requires two separate charts to communicate. One chart contains the symbols, the other the key that allows the communicator to indicate the location of the symbol they wish to communicate. It’s a bit like using a map – you look up the location of a road in your street atlas and it tells you the grid reference on the page.

The chart that is used to communicate the code can itself be accessed in a variety of ways. For example, someone might point directly to the code, eye point to the code on an eye pointing chart, or access the codes through listener mediated scanning.

Coded access is not straightforward, and it will take time to learn. As with encoding, modelling how to access symbols using this method will help everyone to gain confidence and experience in using it.

It is also helpful if the system and how to use it is well documented - perhaps in a Communication Passport.
