Building confidence in play based activities can help both the communication partner and the eye pointing communicator move away from this slightly pressurised approach.

Communication partners are often tempted to move symbols around to see if the individual can still find them. This is a bit like someone coming along and moving all the keys around on your computer keyboard and then asking you to type out what you got up to during the weekend. Try and keep the location of symbols as consistent as possible to enable individuals to build confidence in their communication system.

This is not to say that testing is never appropriate. However, it just needs to be kept out of day-to-day communication. Day-to-day communication should feel like chatting.

Modelling

Modelling is a vital communication partner skill and means using the individual’s communication system while you talk. We sometimes refer to it as point talking i.e. you are pointing with your finger to symbols while you talk,

It would be almost impossible to learn to use French to communicate if you never experienced someone else using it, and the same is true of language represented by symbols. You will find an indepth discussion of modelling and communication partner skills more generally in Using Paper-Based Symbol Systems with Children - Getting Started wtih AAC.

If you are using a communication book that has some sort of index page, it is worth taking the time to start at the beginning of the book so that you are demonstrating how to find the symbols as well as how to use them. If encoding is being used, you may also want to use this yourself by pointing to both the symbol and its corresponding coloured dot or border (or whatever system is being used) so that you are demonstrating how it works.
