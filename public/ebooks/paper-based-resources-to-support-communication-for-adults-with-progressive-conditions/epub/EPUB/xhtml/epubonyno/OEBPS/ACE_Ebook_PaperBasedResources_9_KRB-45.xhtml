Beatrice’s Communication SystemBeatrice uses a bespoke communication system based on combination access – it involves both eye pointing and listener mediated scanning.

You can see Beatrice communicating with Janet using the whiteboard here. You might find it helpful to watch it once, read the explanation below, and then watch it again to fully appreciate how it works, as Beatrice and Janet are so slick at using the system together! In this video, Beatrice is explaining why her communication system was developed:

To communicate using the whiteboard, Beatrice first eye points to the group of black letters that contains her target letter. Her communication partner then establishes if her letter is in the top or bottom row by asking ‘top row?’. If yes, they then use listener mediated scanning to offer the letters in top row, and if no, in the bottom row. However, familiar communication partners are cleverly able to ignore letters that can’t possibly follow on from the previous one selected e.g. in English, the letter ‘K’ never follows the letter ‘D’ within a word so they don’t bother offering that. This really helps to speed along communication. Just occasionally, they can skip the top row question as there is only one possible letter in the block selected.
