Coded Charts

A coded alphabet chart can be another option to consider if pointing to the full twenty-six letters of the alphabet is becoming difficult. You might consider this if you find it hard to refine a point sufficiently to point clearly to twenty-six separate letters, or if you have a limited range of movement.

Coded layouts effectively make use of a grid reference system. Letters, words and phrases are each given a grid reference, and it is this grid reference (or code) that you communicate. This means that you only point to the small set of characters that give the grid reference rather than having to point to the full alphabet chart. The communication partner applies this code to the main chart to work out what you are selecting. It’s a bit like using a road atlas – places are represented in the index as grid references and then located by applying the grid reference to the map.

Communicating in this way requires two separate charts. One chart contains your letters (and words / phrases) and is displayed where both you and your communication partner can see it. The other chart is placed where you can point to it, and contains the grid references that allow you to communicate the location of your letter / word / phrase.

In this example, the first chart shows the letters on offer. The second smaller chart is the one that you point to.

To communicate the letter ‘P’ on the main chart, you would first select the number ‘3’ on your chart of codes, as ‘P’ is in the third row. You would then select the colour ‘blue’ as ‘P’ is in the blue column.
